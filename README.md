## :notebook: サイト概要

　このサイトはセブ島のとある語学学校の先生達とその卒業生達がコミュニケーションを取ることを目的としたコミュニティサイトです。主な機能は１、**投稿機能**　２、**メッセージ機能**　３、**英会話レッスンマッチング**※の３つです。<br>  Twitterなどの有名SNSよりも、より小さなコミュニティのため、密なコミュニケーションを取ることができます。また、利用者が日本人とフィリピン人であるため、言語は**英語**を使用しています。<br>
 ※　英会話レッスンマッチングとは、ユーザーがオンライン英会話レッスンの募集・応募ができる掲示板のこと。<br>
 例）先生が英会話レッスン募集の記事を投稿する　→ その記事を見た生徒が先生のプロフィールページへ行きメッセージを送る。

### :open_file_folder: サイトテーマ
セブ島の語学学校の先生と日本人の卒業生たちが交流できるSNSサイト

### :white_check_mark: テーマを選んだ理由
* セブ島で語学留学をしていた経験があり、帰国後、セブ島の先生たちとコミュニケーションをとることのできる場所がなかったため。
* セブ等の先生たちがコロナで仕事を失っていると聞いたので、何らかの形で協力したいとずっと考えていたため。

###  :two_men_holding_hands: ターゲットユーザ

セブ島に在住している語学学校の先生と日本に在住している卒業生

### :speech_balloon: 主な利用シーン

* ユーザーが写真付きの記事を投稿する。
* ユーザー同士がメッセージを送り合う。
* オンライン英会話レッスンの先生（生徒）を探す。
